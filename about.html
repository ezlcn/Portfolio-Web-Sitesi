Ezgi Laçin

Merhaba, Ben Ezgi Laçin 22 yaşındayım. Çukurova Üniversitesi'nde Bilgisayar Programlama bölümünden mezun oldum. İnnova Bilişim Çözümleri'nde software developer olarak staj yaptım ve bu süreçte birçok projede yer alarak yazılım geliştirme sürecinde deneyim kazandım. Üniversite eğitimimi tamamladıktan sonra Kodluyoruz ve Sabancı Vakfının beraber yürütüğü Yazılımla Güçlenen Kadınlar Front-end kursuna katıldım. Burada Html,css,boostrap,javascrip eğitimleri alarak projeler geliştirdim ve front-end alanında bilgi birikimime katkıda bulundum. Web teknolojileri konusunda temel bilgilere sahibim, özellikle HTML 5, CSS 3, Bootstrap 4, JavaScript ve C# konularında deneyimim var. Ayrıca .NET Framework ile geliştirdiğim projelerim bulunuyor.
